Projects

Apollo Music Player

This is an online music player that takes songs from YouTube or SoundCloud and allows the user to create dynamic, custom playlists. Tech stack includes React.js, including built in state management using useReducer and UseContext. React also has a built in media player called ReactPlayer that was used for this project. Apollo Client was used to connect the front end to a database I set up on Neon, a serverless database host. This Project was built in order to explore and work with GraphQL and RestAPI alternative - the main advantage with GraphQL is that you create custom queries, so that you only get the data that you need instead of a response as a JSON object of information that you don't need. Some drawbacks to GraphQL, are that it brings in some added complexity, so for simple projects GraphQL may be overkill and not necessary.

Face Recognition App

App allows for user login utilizing database to persist information and allow users to keep track of photos they have uploaded into the face recognition app. The technology allows a user to grab the link to any image found on the web featuring a person's face. When the user enters the link to the image the app will display the image and be able to detect where the face is in the picture, and display a blue box around it. The app also tracks how many pictures a user has submitted over time. Project utilizes React.js for front-end and Redux for state management, with and Express.js server taking care of the API calls and running a PostgreSQL database running on Render host site.
